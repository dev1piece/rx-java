## 스터디 진행
* 해당 챕터는 모두 읽고, 독서 토론회 방식으로 진행
* 각 챕터별 코드는 repository에 member별 패키지를 만들었고, 그 하단에 챕터별로 작성한다
  -  ex) src/rhyno/ch1 , test/rhyno/ch1

## Ground Rule
* 매주 월, 목요일 점심에 진행한다
* 업무상 참석을 못하는 경우, 사전통보시 합의
* 연락없이 참석하지 않으면 별금 5천원
* 탈퇴는 모두의 합의시 통과
* 교재 안가져오면 벌금
